# TroopBox AI

TroopBox AI is a modern AI tools directory that helps users discover and access 122+ free AI tools across multiple categories including chatbots, image generation, coding assistants, productivity tools, resume builders, website creators, and more.

## Features

- 122+ AI tools in one place
- Modern and responsive UI
- Fast search functionality
- Category-based tool discovery
- Clean and user-friendly design
- Mobile-friendly experience
- Easy deployment with Vercel

## Categories

- AI Chat
- Local AI Chat
- Image Generation
- Audio Tools
- Coding Assistants
- Productivity Tools
- Presentations
- Resume Builders
- ATS Tools
- Website Builders

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Vercel

## Installation

Clone the repository:

```bash
git clone https://github.com/rohilla2011/TropBox-Ai.git
```

Open the project folder:

```bash
cd TropBox-Ai
```

Run locally:

```bash
open troopbox-ai.html
```

or simply open the file in your browser.

## Deployment

The project can be deployed easily on:

- Vercel
- Netlify
- GitHub Pages
